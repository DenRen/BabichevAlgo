Палиндромы
Палиндром — строка, читающаяся одинаково в любом направлении. Любую строку можно превратить в палиндром, убрав некоторое количество букв.

Ваша задача — определить, сколькими способами можно превратить заданную строку, состоящуюю не более, чем из 30 заглавных букв латинского алфавита, в палиндром. Удаление букв, находящихся в различных позициях и приводящее к одному и тому же палиндрому — подсчитывается раздельно. Удаление пустого множества символов тоже учитывается.

Формат входных данных
Строка не более 30 символов
Формат результата
Количество возможных палиндромов
Примеры
Входные данные
AAAA
    
Результат работы
15
    
Входные данные
ABRACADABRA 
     
Результат работы
131
