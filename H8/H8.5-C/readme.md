Число сочетаний по модулю
Вычислите число сочетаний из N по K по простому модулю P.

Формат входных данных
N K P

1 ⩽ K ⩽ N ⩽ 100000000, P < 1018. P — простое число.

Примеры
Входные данные
10 4 10007
    
Результат работы
210
